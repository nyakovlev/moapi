Installing auditd

Referenced from BookofZeus and page 194 of CIS Ubuntu Linux 16.04 Benchmark

auditd allows system administrators to observe activities going on throughout the day and monitor changes in their systems, such as unauthorized access or modification of data. It is recommended for a secured system to have a logging service (rsyslog) along with a properly configured auditing service.


To install auditd, simply type sudo apt-get install auditd.


That should be it! Further tasks involving auditd configuration are covered in other vulnerability checks.
