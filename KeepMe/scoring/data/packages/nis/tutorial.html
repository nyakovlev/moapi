Removing NIS client

Referenced from page 137 of CIS Ubuntu Linux 16.04 Benchmark

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


NIS is typically used to distribute system configuration files between a client and a server. If not specifically required by the readme, this service should be removed.


To check if NIS client is installed on your system, you can type dpkg -l nis.


If the name of the package is preceded by "ii", then it is installed. To remove it, simply type sudo apt-get purge nis.


That should be it!
