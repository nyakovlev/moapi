Removing Talk Client

Referenced from page 141 of CIS Ubuntu Linux 16.04 Benchmark

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


The talk client is used to send and receive messages between systems using a terminal session. This will likely be unneeded in the competition image, and it should be removed.


To check if talk client is installed on your system, you can type dpkg -l talk.


If the name of the package is preceded by "ii", then it is installed. To remove it, simply type sudo apt-get purge talk.


That should be it!
