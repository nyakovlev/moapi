Removing FTP Server

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


FTP server (vsftpd) allows users to remotely connect to your system to access and manage files. Unless specifically required in the README, this functionality should be removed.


To check if vsftpd is installed on your system, you can type dpkg -l vsftpd.


If the name of the package is preceded by "ii", then it is installed. To remove it, simply type sudo apt-get purge vsftpd.


That should be it!
