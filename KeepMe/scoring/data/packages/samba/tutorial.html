Removing Samba

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


Samba is a software suite that allows Windows-based clients to connect to your machine and access Linux-based resources. Unless specifically required by the README, this program should be removed.


To check if Samba is installed on your system, you can type dpkg -l | grep samba. (the grep command is to find packages with names containing "samba")


If results come up, then Samba is likely installed. To remove Samba, purge the results that return from the above command. (For example: sudo apt-get purge samba samba-common samba-common-bin python-samba samba-dsdb-modules samba-libs samba-vfs-modules)


That should be it!
