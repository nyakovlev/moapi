Removing SSH Server

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


SSH Server allows users to connect to the machine over the connected network and use a command line, in the same way that they would use a terminal on the machine itself. If not specifically required by the README, this functionality should not be made available.


To check if an SSH server is installed on your system, you can type dpkg -l openssh-server.


If the name of the package is preceded by "ii", then it is installed. To remove it, simply type sudo apt-get purge openssh-server.


That should be it!
