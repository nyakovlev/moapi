Removing Wireshark

Installing various packages can make your system vulnerable to any known or discovered security issues in the software. Hackers trying to gain access to your machine often can take advantage of such security holes in your system's software. For this reason, it is highly recommended to purge anything on your machine that is not required to be there.


Wireshark is a utility used to analyze network packets. It is quite useful for debugging complex network issues, but it can pose a network security risk if not absolutely needed on the system.


To check if Wireshark is installed on your system, you can type dpkg -l | grep wireshark. (the grep command is to find packages with names containing "wireshark")


If results come up, then Wireshark is likely installed. To remove it, purge the results that return from the above command. (For example: sudo apt-get purge wireshark wireshark-common wireshark-qt libwireshark8 libwireshark-data)
